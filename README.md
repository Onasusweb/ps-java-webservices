ps-java-webservices
===================

Prestashop java web services    
JAVA web services API for Prestashop

Prestashop depuis sa version 1.4 offre nativement une API services web. La documentation propose une librairie PHP.    
Ce projet offre un framework similaire pour le developpement d'applications Prestashop en JAVA. Je l'ai déjà utilisé dans le cadre de quelques projets expérimentales.
